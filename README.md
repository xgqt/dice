# dice


## About

A small utility to roll some dice


## Usage

```fundamental
dice [OPTION]... [NUMBER]
```


##  Options
- -h                 - Print short help
- --help             - Print long help
- -v     / --version - Print version number
- -s     / --small   - Roll small dice
- -0     / --none    - Only generate numbers


## Exmaples

```bash
./dice
```

Output:

```fundamental
 _______
| o     |
|   o   |
|     o |
'-------'
```
